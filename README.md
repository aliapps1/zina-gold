# ZINA Gold Card Demo

Professional Web App demo for **ZINA Gold**, a gold-backed digital account and card concept for UAE/GCC investor presentations.

## Files

- `index.html` — Main app structure
- `styles.css` — Visual design and responsive layout
- `app.js` — Language switching, card status, conversion simulation
- `manifest.json` — PWA metadata
- `icon.svg` — App icon

## Important Legal Note

This is a **concept demo only**. It is not a live financial service. Banking, card issuing, custody, KYC, AML, and ATM functionality require regulated partner integration and approvals.

## GitHub Pages Setup

1. Create a GitHub repository, for example: `zina-gold-web-app`
2. Upload all files in this folder to the repository root
3. Go to **Settings → Pages**
4. Choose **Deploy from branch**
5. Select branch: `main`, folder: `/root`
6. Save and wait for GitHub Pages link

## Languages

The demo supports:

- English
- Arabic

The language can be switched from the top navigation.
